# Bowling League Web App

Full-stack web application displaying bowler information for the Marlins and Sharks bowling teams.

## Tech Stack
- **Backend**: ASP.NET Core Web API with Entity Framework Core + SQLite
- **Frontend**: React + TypeScript + Vite

## Running the Application

### Backend
```bash
cd backend
dotnet run
```
API runs at http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs at http://localhost:5173
